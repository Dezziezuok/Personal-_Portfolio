FinTech Site

I participated in the UA Innovate Hackathon 2025, where I collaborated with a team of four to develop a PNC Bank Fintech site.

The prompt involved creating a user-friendly interface for financial advice for young people in their 20s and implementing financial features.

I used HTML, CSS, and JavaScript to build the front-end of a website that provides personalized financial advice and resources for young adults, helping them make informed decisions about their finances.

It was my first user interface project, and I learned a lot about the importance of user experience and design in software development.
